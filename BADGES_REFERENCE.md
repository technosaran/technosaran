## üîß Customization Tips

### Change Theme Globally
Replace `theme=tokyonight` with any theme name in all stat cards.

### Adjust Card Height
Add `&height=170` to stat card URLs.

### Hide Specific Stats
Add parameters like:
- `&hide=stars` - Hide stars
- `&hide=commits` - Hide commits
- `&hide=prs` - Hide PRs
- `&hide=issues` - Hide issues

### Show Private Contributions
Add `&count_private=true` to GitHub stats.

### Custom Colors
Add `&title_color=00eaff&text_color=ffffff&bg_color=0d1117` for custom colors.
